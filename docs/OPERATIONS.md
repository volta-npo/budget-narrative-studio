# Budget Narrative Studio Operations Manual

## What this product does

A structured builder for grant budget narratives that match numbers to purpose.

## Primary workflow

1. Complete the purpose-built domain tool section.
2. Attach evidence in the evidence locker.
3. Complete the v1 workbench: Budget Narrative Studio.
4. Resolve readiness warnings.
5. Certify release gates only with real project evidence.
6. Export JSON, CSV, Markdown, and print packets for the project folder.

## Domain outputs

- Budget CSV
- Budget narrative
- Reviewer comments report

## Privacy and retention

Data is local-first. Export intentionally, store securely, and delete browser state after handoff if working on a shared device.


## SaaS pilot operations

Use the standalone SaaS command center to prepare Budget Narrative Studio for paid pilots and chapter-scale operations. Review account ownership, segment, plan, operating metrics, playbook status, automations, governance evidence, and roadmap phase before presenting the workspace as a standalone SaaS. Export the SaaS brief and operations CSV alongside the standard handoff packet.
