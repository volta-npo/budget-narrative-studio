# Budget Narrative Studio v1.0 Product Specification

## Workbench

Budget Narrative Studio

## Domain table

Budget line items

## Metrics

- Narrative Alignment
- Math Accuracy
- Cap Compliance

## Domain rows

- Line items entered
- Quantity/unit cost calculated
- Funding source assigned
- Justification written
- Activity linked
- Admin cap checked
- Reviewer comments resolved
- Narrative exported

## Generated deliverables

- Budget CSV
- Budget narrative
- Reviewer comments report

## Validation gates

- Totals must match request
- No negative amounts
- Every line needs calculation and justification
